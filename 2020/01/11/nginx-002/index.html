ngixn搭建静态服务器

搭建静态服务器(完整配置文件)

root 与 alias的不同

root 会将匹配到的路径与root路径拼装成完整路径
如 location /abc/ { root $root/;} /abc/a.html => 返回 $root/abc/a.html
alias 会替换匹配到的路径，alias路径必须闭合, 如 location /abc/ { root $root/;} /abc/a.html => 返回 $root/a.html

限速

通过内置变量 limit_rate实现,如 limit_rate 1k,此时以每秒1k的速度 向客户端传输数据

日志

指令 log_format main 日志格式 //指定 日志格式变量名 日志格式
使用 access_log /logs/access_log main
